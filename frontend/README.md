# 🧠 Life Skills App – Frontend

This is the **frontend** for the Soft Engineering Project – May 2025. It’s a Vue 3 + Vite-based application designed to help children build essential life skills through interactive modules, AI feedback, and gamified badges.

---

## 🚀 Project Setup

### 1. Clone the Repository (optional)

```bash
git clone -b frontend https://github.com/kohliaryan/soft-engg-project-may-2025-se-May-10.git
```
```bash
cd soft-engg-project-may-2025-se-May-10
```

---

### 2. Go into the folder `Team-10_Frontend_Milestone_3`

```bash
cd Team-10_Frontend_Milestone_3/
```

---

### 3. Install Dependencies

```bash
npm install
```

---

### 4. Run the Development Server

```bash
npm run dev
```

---

This will start the **app** at:

**http://localhost:5173**
